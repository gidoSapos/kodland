Esse foi meu primeiro projeto mexendo com pygame zero, apesar de ter uma experiencia solida com python não vou mentir que foi desafiador aprender sobre essa biblioteca. 
Tive alguns desafios como:
- a biblioteca não aceita .mp3,
- não consegui dividir o projeto em arquitetura de pastas, apesar de não querer ter apenas o main.
- colisões e plataformas, apesar de ser uma fisica básica, foi um pouco sofrido fazer isso sem nunca ter pego um projeto desses antes.
- eu tava tentando fazer uma condicional pra tirar todo o volume mas decidi que fazer uma função seria mais fácil. Mas até descobrir me levou um tempo.
Enfim, o jogo não está a coisa mais linda do munod, está bem básico, mas espero que com a Kodland eu possa polir e refinar essas habilidades que tenho com python mas aplicar na biblioteca pgzero.
